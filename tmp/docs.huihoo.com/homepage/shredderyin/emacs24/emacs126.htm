Summary

Now that you have been introduced to the concepts of Ediff, you might find that you never want to use the command-line versions of diff or patch anymore. Furthermore you might find that you can work with several people on a project, each having a copy of the file, and then use Ediff to merge each member's work together. (If you do this much, you should, however, consider using the program package called CVS.)

If you find that you frequently use Ediff, you should read the documentation which comes with Ediff. It is available through the info pages.
