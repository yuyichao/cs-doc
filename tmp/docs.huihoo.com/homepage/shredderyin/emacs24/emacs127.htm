Q&A

Q Why isn't the current difference in both buffer A and buffer B highlighted with the same color?

A To be honest, I don't know. But a guess is that the people who made Ediff were so happy about colors that they simply had to use different colors. Fortunately, you have learned to use customize, and Ediff's colors are customizable, so please change them at your will.

Q How can Ediff help me patch files?

A It will let you step through the patches one by one, as if you have asked Emacs to diff the original file, and the patched file.

Q I use GNU Emacs version 19, and I really need to increase the value of the variable ediff-auto-refine-limit. Unfortunately you say that this doesn't work in GNU Emacs version 19. Is there something I can do?

A Yes. On the CD-ROM accompanying this book is a newer version of Ediff (version 2.67). You can replace the one that came with Emacs with this one. For this to work, however, you also need to install the customize library. Maybe you should consider whether this is the time to install Emacs 20.
