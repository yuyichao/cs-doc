Summary

It should now be clear that Emacs can support a vast array of system interactions, including running external programs synchronously, asynchronously, and interactively. In addition, Emacs has complete control over the filesystem with the capability to be a file manager that can move and manipulate files. You learned how to print from Emacs using the convenience functions at any level of detail that might be needed.

Lastly, it should now be possible to write simple programs that add commands into Emacs to perform specialized shell commands and specialized printing.
