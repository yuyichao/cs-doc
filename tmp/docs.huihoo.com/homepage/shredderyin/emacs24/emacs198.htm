Keyboard and Events

In addition to your modifier and character keys, there are function keys, the keypad set, and possibly other keys on the keyboard. These keys don't generate characters, but named events. For example, the function keys and keypad keys generate event names: 


f1,f2,f3 .. f20 kp-1, kp-2, ... kp-9, kp-add, kp-enter, kp-substract, kp-multiply, kp-divide
