Exercises

1. Find out what is bound to the key C-a and find the documentation for it--both the short one and the info pages.

2. Find the info pages on bookmarks, described in Hour 8, "Searching for Text in Multiple Files." Is there an Emacs command that locates point on a specific line? If yes, is it bound to a key?

3. Find the customization page for customize and see whether there is anything that you want to change.

4. C-k erases the text to the end of the line but does not remove the line break itself. Try to change this so Emacs will remove the line break too.
